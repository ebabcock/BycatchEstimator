# Function to make data summarizes including ratio estimate at

stratification defined by strataVars. No pooling for missing strata

## Usage

``` r
MakeSummary(obsdatval, logdatval, strataVars, EstimateBycatch, startYear)
```

## Arguments

- obsdatval:

  Value

- logdatval:

  Value

- strataVars:

  Value

- EstimateBycatch:

  Value

- startYear:

  Value
