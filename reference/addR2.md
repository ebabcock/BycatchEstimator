# Return a column of R squared values from input of a MuMin dredge table

Return a column of R squared values from input of a MuMin dredge table

## Usage

``` r
addR2(dredgeTable, obsdatval, funcName)
```

## Arguments

- dredgeTable:

  dredge output from MuMin

- obsdatval:

  observer data

- funcName:

  function used in fitting (e.g. glm or glmmTMB)
