# Function for Pennington(1983) method

Output is result of Gm(t) function for use in calculations of the design
based delta estimator

## Usage

``` r
Gm(m, t, jmax = 10)
```

## Arguments

- m:

  m is the number of non-zero values

- t:

  t is a real number input

- jmax:

  jmax is the upper limit of an infinute sum used in calculations. 10 is
  usually sufficient
