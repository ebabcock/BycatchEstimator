# Function to simulate DHARMa residuals from a negative binomial GAM or GLM

Function to simulate DHARMa residuals from a negative binomial GAM or
GLM

## Usage

``` r
simulateNegBinGam(modfit, nsims = 250, offsetval = 1)
```

## Arguments

- modfit:

  Value

- nsims:

  Value

- offsetval:

  Value
