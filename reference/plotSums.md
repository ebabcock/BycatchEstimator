# Function to plot either total positive trips (binomial) or total catch/bycatch (all other models)

Function to plot either total positive trips (binomial) or total
catch/bycatch (all other models)

## Usage

``` r
plotSums(
  yearpred,
  modType,
  fileName,
  subtext = "",
  allVarNames,
  startYear,
  common,
  run,
  catchType,
  catchUnit,
  VarCalc
)
```

## Arguments

- yearpred:

  Value

- modType:

  Value

- fileName:

  Value

- subtext:

  Value

- allVarNames:

  Value

- startYear:

  Value

- common:

  Value

- run:

  Value

- catchType:

  Value

- catchUnit:

  Value
